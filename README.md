  https://chrome.google.com/webstore/detail/json-viewer/gbmdgpbipfallnflgajpaliibnhdgobh

* Xdebug helper

  配合Phpstorm使用

  https://chrome.google.com/webstore/detail/xdebug-helper/eadndfjplgieldjbigjakmdgkmoaaaoc

  浏览器代理，可以实现在全局和pac中自动切换
